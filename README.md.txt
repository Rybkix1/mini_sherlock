# 🔎 Mini_Sherlock

Mini_Sherlock is a lightweight OSINT tool written in Python, inspired by the Sherlock project.

It allows you to check the availability of usernames across various websites.

---

## 🚀 Features

- Scans usernames across multiple websites
- Quickly checks account availability
- Simple CLI interface
- Fast version (V2) with threading

---

## ⚡ Versions

### 📦 V1
- Basic version
- Slower, sequential checking

### 🚀 V2 (FAST)
- Faster scanning (threading)
- Improved detection:
  - FOUND
  - NOT FOUND
  - BLOCKED / UNKNOWN

---

## 📥 Installation

Install the required libraries:

pip install requests

⚠️ Note

The tool works based on public websites.

It does not guarantee 100% accuracy of results, as some sites block requests or return different responses.
